# ダイアログの作成
まずは「ダイアログのUI」を作りましょう。新規`Eiga`ノードを作成してください。

最低限会話表示のテキストボックス(`RichTextBox`など)があれば動作します。

# イベントの接続
`Eiga`ノードには以下のイベントがあります。

| イベント名 | 詳細 |
| ---- | ---- |
| `add_text(text: String)` | テキストボックスに追加される1文字が渡されます |
| `init_text()` | テキストボックスの初期化時に呼ばれます |
| `scene_trans` | `->`によって指定されたシーンが渡されます |
| speaker_changed(speaker: String) | 話者が渡されます |
