# eiga
このプラグインは動きのある会話シーンを作ることを手助けする独自言語、`EigaLang`の **「インタプリタ」** です。

# 説明
`EigaLang`は **「どのセリフ」の「どのタイミング」で「どんな処理をしたいのか」をテキストベースで簡潔に指定する** ことを目的としています。「会話中に特定の動作をさせる」という実装をするのは、意外と大変です。この言語を使うことで、ゲーム開発における会話シーンがより簡単に動きのあるものになります。
