# eiga
このプラグインは動きのある会話シーンを作ることを手助けする独自言語、`EigaLang`の **「インタプリタ」** です。

# 目的
`EigaLang`は **「どのセリフ」の「どのタイミング」で「どんな処理をしたいのか」をテキストベースで簡潔に指定する** ことを目的としています。 テキストボックスの管理などを皆さんに任せ情報の橋渡しに徹することで、よりユニークで楽しい会話シーンを作ることが出来ると考えています。
